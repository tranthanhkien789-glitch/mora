Quy ước đặt tên

Naming convention · Version control · Archive

Format

[PRODUCT]-[CATEGORY]-[NUMBER]_[descriptive-name]-v[version].[ext]
Ví dụ: MORA-BRD-001_brand-guidelines-v2.html
Đọc: Mora → Brand → File 001 → Brand Guidelines → Version 2 → HTML
Quy tắc bổ sung
Version: v1, v2... cho major. v1.1 cho minor.
Thư mục con: lowercase, gạch ngang (sops/, checklists/, contracts/)
Không duplicate: Mỗi file 1 vị trí. Cross-reference thay vì copy.
