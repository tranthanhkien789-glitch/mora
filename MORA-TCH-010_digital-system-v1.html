Caption rules

Luôn luôn
✓ Ít nhất 1 chi tiết cụ thể (tên, nguồn gốc, lý do)
✓ Tiếng Việt trước, Anh sau nếu bilingual
✓ Natural light photography tại đúng store space
✓ Show process — hands, tools, craft visible
Không bao giờ
× Emoji spam trong caption
× Countdown timer / "Chỉ còn X suất"
× Studio trắng isolated product shot
× Caption chỉ có emoji, không có nội dung
C03 · Brand guidelines digital

Digital Guidelines

CSS variables, type scale, và component rules cho mọi digital touchpoint của Mora. Copy vào mọi project. Không được hardcode màu sắc hay font ngoài hệ thống này.

CSS variables — copy vào mọi project

/* Mora Digital Design System */ :root { /* Colours */ --mora-ink: #1A1916; /* Primary text */ --mora-teal: #0F6E56; /* Primary brand */ --mora-teal-dark: #085041; /* Hover / pressed */ --mora-teal-light: #E1F5EE; /* Backgrounds */ --mora-amber: #854F0B; /* Accent / emphasis */ --mora-amber-light: #FAEEDA; /* Amber backgrounds */ --mora-cream: #FAF9F6; /* Page background */ --mora-linen: #E8DFD0; /* Borders / dividers */ --mora-warm-600: #8C7A62; /* Muted text */ --mora-blue: #185FA5; /* Digital links / UI only */ /* Fonts */ --font-display: 'Cormorant Garamond', Georgia, serif; --font-body: 'DM Sans', system-ui, sans-serif; --font-mono: 'DM Mono', monospace; }

Discovery Quiz

Digital seed experience — convert người chưa đến thành người muốn đến. 5 câu hỏi về lifestyle, không phải về kỹ thuật hương. Kết quả: nhóm hương phù hợp + CTA booking.

Gỗ tươi mát
Bạn hướng đến sự cân bằng — không quá ngọt, không quá nặng. Nhóm hương gỗ tươi mát (Woody Fresh) sẽ đồng hành tốt trong hầu hết các tình huống, từ buổi sáng làm việc đến buổi tối thư giãn.
Gợi ý: Cedarwood · Vetiver · Ambroxan · Bergamot

5 câu hỏi — logic

Q1 · Occasion
Khi nào mang? → maps to sillage
Q2 · Environment
Thích không khí nào? → maps to fragrance family
Q3 · Personality
Từ mô tả bạn? → maps to character
Q4 · Intensity
Cường độ mùi? → maps to concentration
Q5 · Priority
Điều quan trọng nhất? → confirm direction
Kết quả map ra 4 fragrance families: Woody Fresh · Floral · Citrus Fresh · Oriental Woody
